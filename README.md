## <div id="字幕プロジェクトのポリシー">📌 字幕プロジェクトのポリシー</div>
- 使用音楽や台詞に著作権上の問題がないこと
  - ワールド制作者が著作者、または著作者の許可がある場合はOK
- ワールドが常設であること
  - 字幕起こしや、テストを十分な回数行える場合はOK
- 再生ギミックはユーザー操作や定時で発動すること
